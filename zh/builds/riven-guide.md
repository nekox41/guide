# 紫卡指南

## 主要武器 - 天穹之顶

双词条

1. [**多重 暴伤 负面**](https://warframe.market/auctions/search?type=riven\&positive_stats=critical_damage,multishot\&negative_stats=has\&sort_by=price_asc\&weapon_url_name=zenith) ⇨ 3 多重 单人/双人 配置\
   在灵活性方面，这可能是最好的选择。
2. [**暴率 暴伤 负面**](https://warframe.market/auctions/search?type=riven\&positive_stats=critical_chance,critical_damage\&negative_stats=has\&sort_by=price_asc\&weapon_url_name=zenith) ⇨ 2 多重单人配置 ⇨ 替换关键延迟\
   不挑数值，而且很便宜。
3. [**多重 基伤 负面**](https://warframe.market/auctions/search?type=riven\&positive_stats=multishot,base_damage_%2F_melee_damage\&negative_stats=has\&sort_by=price_asc\&weapon_url_name=zenith) ⇨ 3 多重 单人/双人 配置\
   比起多重暴伤的稍逊一筹。

**三词条**

1. [**多重 暴伤 射速 负面**](https://warframe.market/auctions/search?type=riven\&positive_stats=multishot,fire_rate_%2F_attack_speed,critical_damage\&negative_stats=has\&sort_by=price_asc\&weapon_url_name=zenith) ⇨ 2 多重双人配置 ⇨ 替换射速
2. [**多重 基伤 射速 负面**](https://warframe.market/auctions/search?type=riven\&positive_stats=multishot,base_damage_%2F_melee_damage,fire_rate_%2F_attack_speed\&negative_stats=has\&sort_by=price_asc\&weapon_url_name=zenith) ⇨ 2 多重双人配置 ⇨ 替换射速

{% hint style="warning" %}
至少需要 100% 多重射击，这是必须的。

这些紫卡通常是大材小用的，而且非常昂贵，但是它们可以让你达到10射速，以 2ms 的延迟射击。
{% endhint %}

## 近战 - 瓦斯提枪刃

{% hint style="warning" %}
只有在你计划使用**次要爆发**的时候，才需要购买紫卡。瓦斯提枪刃的紫卡很少，所以通常很贵。
{% endhint %}

**双词条**

1. **攻击速度 额外连击几率 负面** ⇨ 经典配置\
   额外连击几率允许你快速获取连击数，以便于配合[**次要爆发**](https://warframe.huijiwiki.com/wiki/%E6%AC%A1%E8%A6%81%E7%88%86%E5%8F%91)。\
   由于配卡中已经有了很多攻击速度，所以攻击速度不是必须的。
2. **初始连击 额外连击几率 负面** ⇨ 经典配置\
   更高的初始连击数，可以更快达到 12x 连击。
3. **初始连击 攻击速度 负面** ⇨ 经典配置\
   更高的初始连击数，可以更快达到 12x 连击。

**三词条**

* **攻击速度 额外连击几率 初始连击 负面** ⇨ 经典配置\
  完全是大材小用而且超级贵，如果你有的话就用吧。
