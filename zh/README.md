# 夜灵指南

随时欢迎加入我们的 Discord [**Eidolon Server**](https://discord.com/invite/mQhzbVv5bg)

## **摘要**

本指南适用于所有水平的玩家：刚入门的玩家和想要更进一步的玩家。\
它的目标是帮助刚入门的玩家理解基本机制，以及为想要参与竞速的玩家提供帮助并改善成绩。
